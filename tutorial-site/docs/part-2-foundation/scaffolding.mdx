# Chapter 3: Project Scaffolding

> Time to create the project and lay the foundation for everything that follows.

By the end of this chapter, you'll have a running React + TypeScript project with all dependencies installed and your project structure organized.

---

## What You'll Do

- Scaffold a new project with `npm create vite@latest`
- Install React, TypeScript, Tailwind CSS v4, Firebase, Capacitor, and Lucide React
- Set up the folder structure
- Write your TypeScript interfaces (`types.ts`)
- Configure environment variables
- See your first `npm run dev` output
